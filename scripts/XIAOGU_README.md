# 小股股票分析提醒脚本

基于国内免费公开财经接口（新浪财经/东方财富）的股票提醒脚本，替换原有的"小股早盘提醒"和"小股午盘提醒"任务。

## 功能特性

- 📊 **大盘指数行情**: 获取上证指数、深证成指、创业板指、沪深300等实时行情
- 🔥 **板块涨幅榜/跌幅榜**: 获取行业板块排行和领涨股
- 💰 **北向资金流向**: 获取沪深港通资金流向数据
- 📈 **市场概况**: 涨跌家数、涨停跌停统计
- 💡 **智能分析**: 自动生成"小股"风格的专业分析内容
- 🚀 **自动发送**: 运行后自动发送到指定Feishu群

## 技术特点

- ✅ 完全免费，无需API Key
- ✅ 多接口降级方案，稳定性高
- ✅ 自动重试机制（3次重试）
- ✅ 收盘后自动显示"数据获取中"
- ✅ 符合"小股"专业分析风格

## 使用方法

### 命令行运行

```bash
# 生成并发送早盘报告
python3 scripts/xiaogu_stock_report.py --type morning

# 生成并发送午盘报告
python3 scripts/xiaogu_stock_report.py --type noon

# 仅生成报告，不发送（测试模式）
python3 scripts/xiaogu_stock_report.py --type morning --dry-run
```

### 测试脚本

```bash
# 测试早盘报告（不发送）
./scripts/test_xiaogu.sh morning

# 测试午盘报告（不发送）
./scripts/test_xiaogu.sh noon
```

## 定时任务配置

任务已配置在 `~/.openclaw/cron/jobs.json` 中：

| 任务 | 执行时间 | 命令 |
|------|----------|------|
| 小股早盘提醒 | 周一到周五 9:20 | `python3 xiaogu_stock_report.py --type morning` |
| 小股午盘提醒 | 周一到周五 12:50 | `python3 xiaogu_stock_report.py --type noon` |

## 数据来源

| 数据类型 | 主要来源 | 降级来源 |
|----------|----------|----------|
| 大盘指数 | 新浪财经 hq.sinajs.cn | 东财行情接口 |
| 板块排行 | 东方财富板块接口 | 概念板块接口 |
| 北向资金 | 东方财富资金流向 | 历史数据接口 |
| 市场概况 | 东方财富涨跌统计 | 简单统计接口 |

## Feishu群配置

默认发送到群ID: `oc_68a232050ed9787d0245ae6929b1d429`

如需修改，请编辑脚本中的 `FEISHU_GROUP_ID` 变量。

## 故障排查

### 数据获取失败

- 收盘后部分API可能返回空数据，这是正常的
- 脚本会自动使用降级方案
- 非交易时间会显示"数据获取中"

### 发送失败

- 检查Feishu群ID是否正确
- 检查openclaw配置是否正常
- 查看日志: `~/.openclaw/logs/`
